**React Router Dom** es la librería estándar para gestionar la navegación en aplicaciones React. Permite definir rutas declarativamente y navegar entre vistas sin recargar la página.

## Cuándo usarlo

Usa React Router Dom cuando tu aplicación tiene múltiples vistas o páginas que el usuario puede navegar. Si usas Next.js, no lo necesitas — Next.js tiene su propio sistema de rutas basado en el sistema de archivos.

## Instalación

```bash
pnpm add react-router-dom
```

## Conceptos clave

| Concepto | Descripción |
|---|---|
| `BrowserRouter` | Proveedor que habilita el routing en la app |
| `Routes` | Contenedor de las definiciones de ruta |
| `Route` | Asocia una URL a un componente |
| `Link` | Navegación declarativa sin recarga |
| `useNavigate` | Navegación programática |
| `useParams` | Acceso a parámetros de la URL |
| `Outlet` | Renderiza rutas hijas en layouts anidados |

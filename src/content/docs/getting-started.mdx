**Sanghel Playbook** es un CLI interactivo que scaffoldea proyectos o instala patrones en proyectos existentes. Todo desde la terminal, sin copiar código manualmente.

---

## Prerrequisitos

- Node.js 18 o superior
- Un gestor de paquetes: `pnpm`, `npm` o `yarn`

---

## Paso 1 — Ejecutar el CLI

```bash
npx sanghel-playbook
```

No necesitas instalar nada globalmente. `npx` descarga y ejecuta el CLI en el momento.

---

## Paso 2 — Elegir el modo

Al ejecutar el comando aparece una TUI interactiva con dos opciones:

```
┌ ¿Qué quieres hacer?
● Crear nuevo proyecto
○ Añadir a proyecto existente
```

- **Crear nuevo proyecto** — scaffoldea desde cero con el stack que elijas.
- **Añadir a proyecto existente** — instala patrones del catálogo sobre un proyecto que ya tienes.

---

## Paso 3 — Seleccionar el stack

Si elegiste "Crear nuevo proyecto":

```
┌ Elige el stack base:
● Next.js
○ React + Vite
○ Astro
```

El CLI usa las herramientas oficiales de cada framework (`create-next-app`, `create vite`, `create astro`) para generar el proyecto base.

---

## Paso 4 — Elegir integraciones

Una vez creado el proyecto base, el CLI te muestra las integraciones disponibles para ese stack:

```
┌ Integraciones para Next.js:
✓ React Query
✓ Zustand
  NextAuth v5
  i18next
```

Selecciona con `espacio` y confirma con `enter`. El CLI detecta tu gestor de paquetes automáticamente.

---

## Paso 5 — Resultado

El CLI instala los archivos de código (hooks, componentes, schemas), las guías en markdown y las dependencias npm necesarias.

```
✓ Proyecto listo en ./mi-app
```

Para arrancar:

```bash
cd mi-app
pnpm dev
```

---

## Añadir a proyecto existente

Si elegiste "Añadir a proyecto existente", el CLI te muestra el catálogo por categoría:

```
┌ Categoría:
● React
○ Next.js
○ Astro
```

Navega, selecciona los patrones que necesitas, y el CLI los instala en tu proyecto actual.

---

## Explorar la documentación

Cada patrón tiene su documentación con contexto, ejemplos y decisiones de diseño.

Usa el sidebar para navegar por categoría o busca con `⌘K`.
